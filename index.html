臨床鑑別診断ツール集

臨床所見・検査所見から系統的に鑑別診断を行うインタラクティブツール

🔍
該当するツールが見つかりません
🫘
急性腎障害（AKI）鑑別診断ツール
緊急度評価、尿路閉塞除外、循環動態評価、腎毒性物質確認、尿検査所見、臨床所見・病歴から 腎前性・腎性（ATN/AIN/RPGN/TMA）・腎後性の13疾患を系統的に鑑別します。
6ステップ / 13疾患
🫧
全身浮腫 鑑別診断ツール
浮腫の性状（圧痕性/非圧痕性）、分布、頸静脈怒張、検査所見、随伴症状、薬剤歴から 心不全・肝硬変・ネフローゼ・甲状腺機能低下症など9疾患を系統的に鑑別します。
6ステップ / 9疾患
🧪
低ナトリウム血症 鑑別診断ツール
血清浸透圧、体液量評価、尿浸透圧、尿中Na、検査所見、臨床所見・病歴、薬剤歴から SIADH・体液量減少性・心不全・副腎不全など14疾患を系統的に鑑別します。
7ステップ / 14疾患
💪
無症候性高CK血症 鑑別診断ツール
EAN 2024ガイドライン準拠。性別・人種別ULN基準によるCK評価、神経学的診察、後天性原因除外、 GAA-DBS検査、EMG/NCS、代謝検査、筋MRI、NGSまで12ステップで系統的に鑑別します。
12ステップ / EAN 2024準拠
⚡
低カリウム血症 鑑別診断ツール
重症度評価、偽性/細胞内シフト除外、尿中K排泄評価、血圧・酸塩基平衡から 腎性/腎外性喪失、原発性アルドステロン症・RTA・Gitelman症候群など系統的に鑑別します。
7ステップ / 分岐型アルゴリズム
🔥
高ナトリウム血症 鑑別診断ツール
体液量評価、尿浸透圧、尿量、検査所見、病歴・薬剤歴、DDAVP試験から 中枢性/腎性尿崩症、浸透圧利尿、腎外性水分喪失、飲水不足、Na過剰負荷などを系統的に鑑別します。
7ステップ / 10疾患鑑別
🧪
高カルシウム血症 鑑別診断ツール
PTHを軸にした系統的鑑別。症状・経過、PTH値、尿中Ca/CCCR、ビタミンD/PTHrP、背景疾患、薬剤歴から PHPT・FHH・悪性腫瘍・肉芽腫性疾患・ビタミンD中毒・長期臥床など14疾患を鑑別します。
7ステップ / 14疾患鑑別
🩺
腹痛 鑑別診断ツール
疼痛部位、発症様式、疼痛性状、バイタルサイン・身体所見、随伴症状、リスク因子、検査所見から 急性虫垂炎・胆嚢炎・膵炎・腸閉塞・腸間膜虚血・異所性妊娠など13疾患を系統的に鑑別します。
7ステップ / 13疾患鑑別
🩸
高BUN血症 鑑別診断ツール
BUN/Cr比の自動計算、脱水評価、消化管出血の確認、尿路閉塞評価、薬剤・食事歴、基礎疾患から 腎前性・腎性・腎後性・腎外性の4カテゴリーを系統的に鑑別します。
7ステップ / 4カテゴリー鑑別
🟣
両側視床FLAIR高信号 鑑別診断ツール
MRI画像パターン（DWI/SWI/造影）、発症様式、臨床症状、背景・既往歴、検査所見から 脳深部静脈血栓症・Percheron動脈梗塞・Wernicke脳症・CJD・自己免疫性脳炎など14疾患を系統的に鑑別します。
6ステップ / 14疾患鑑別
🦶
感覚性ポリニューロパチー 鑑別診断ツール
分布パターン、発症様式、電気生理検査（軸索/脱髄）、臨床症状、背景・既往歴、検査所見から 糖尿病性・CIDP・GBS・血管炎性・CMT・FAP・POEMS症候群など25疾患を系統的に鑑別します。
7ステップ / 25疾患鑑別
🔎
ポリニューロパチー鑑別・検査ワークアップツール
発症様式、分布パターン、電気生理検査（軸索/脱髄）、障害モダリティ、随伴症状、背景・既往歴、検査所見から 糖尿病性・CIDP・GBS・血管炎性・POEMS・アミロイドーシスなど24疾患を鑑別し、推奨検査リストを自動生成します。
7ステップ / 24疾患鑑別 + 検査リスト生成
🩺
二次性高血圧 鑑別診断ツール
高血圧の重症度、発症年齢、随伴症状、身体所見、基本検査、リスク因子、精密検査から 原発性アルドステロン症・褐色細胞腫・クッシング症候群・腎血管性・OSAなど12疾患を系統的にスクリーニングします。
7ステップ / 12疾患鑑別
👁️
外転神経麻痺 鑑別診断ツール
麻痺の特徴（片側/両側）、発症様式、疼痛評価、神経学的検査、背景・リスク因子、画像・検査所見から 虚血性・腫瘍性・炎症性・感染性・頭蓋内圧亢進・重症筋無力症など19疾患を系統的に鑑別します。
6ステップ / 19疾患鑑別
🤕
頭痛鑑別診断ツール
発症様式・Red Flags（SNNOOP10）・持続時間・性状・随伴症状・眼症状・増悪因子の7ステップで、 危険な二次性頭痛を先に拾い、一次性頭痛を積極的に診断します。ICHD-3・NICE CG150・ACR 2023準拠。
7ステップ / 22疾患鑑別
🌀
めまい鑑別診断ツール
症候群分類（AVS/t-EVS/s-EVS/慢性）を入口に、危険徴候・眼振所見・聴覚症状・片頭痛関連・慢性特徴・背景因子の7ステップで、 後方循環脳卒中を見逃さず、前庭神経炎・BPPV・前庭性片頭痛・メニエール病・PPPDなど13疾患を系統的に鑑別します。
7ステップ / 13疾患鑑別
🚨
意識障害 鑑別診断ツール
発症様式、バイタルサイン、巣症状、瞳孔所見、運動反応パターン、随伴症状、背景・既往歴、経過の8ステップで、 脳出血・SAH・NCSE・代謝性脳症・薬物中毒・髄膜炎など20疾患を「4本柱」フレームワークに基づき系統的に鑑別します。
8ステップ / 20疾患鑑別
🧠
CCEEG 薬剤調整判断サポートツール
持続脳波モニタリング（CCEEG）の所見に基づき、抗てんかん発作薬の調整判断を支援します。 ESz/ESE判定、周期性放電（LPDs/GPDs/BIPDs）、RDA、IICの評価をACNS 2021用語に準拠してガイドします。
分岐型フローチャート / ACNS 2021準拠
🌡️
不明熱（FUO）鑑別診断ツール
病型分類、発熱パターン、身体所見(PDC)、検査所見、曝露歴、臓器症状、既施行検査の7ステップで、 結核・心内膜炎・リンパ腫・Still病・GCA/PMR・薬剤熱・HLH/MASなど16疾患を4病因群に基づき系統的に鑑別します。
7ステップ / 16疾患鑑別
💔
胸痛 鑑別診断ツール
安定性、発症様式、痛みの性質、バイタルサイン、身体所見、随伴症状、背景因子の7ステップで、 ACS・PE・AAS・心膜炎・気胸・肺炎など14疾患を6カテゴリに基づき並列除外する実践的アプローチ。
7ステップ / 14疾患鑑別
🫁
呼吸困難 鑑別診断ツール
重症度、時間経過、バイタルサイン、呼吸器所見、循環器所見、随伴症状、背景因子の7ステップで、 肺塞栓・気胸・心不全・喘息/COPD増悪・肺炎・ACS・神経筋性呼吸障害など18疾患を5病態群に基づき系統的に鑑別します。
7ステップ / 18疾患鑑別
🔍
神経障害部位推測ツール
MMT（徒手筋力検査）の結果から、障害されている脊髄レベル・末梢神経を推測します。 各筋の神経根レベル重み付けに基づくスコアリングで、上肢(C5-T1)・下肢(L1-S2)・統合解析に対応。
上肢23筋 / 下肢16筋 / 重み付けスコアリング
🧠
多発脳神経障害 局在診断ツール
症状チェックリストから障害脳神経を特定し、病変の解剖学的局在（海綿静脈洞・CPA・頸静脈孔など8部位）を カバレッジ率に基づくスコアリングで推測します。散在的パターンのRed Flag警告機能付き。
14症状 / 8局在 / カバレッジ分析
🤲
振戦 鑑別診断ツール
振戦の出現タイミング・左右差・リズム・随伴所見・薬剤歴から、本態性振戦・パーキンソン病・ジストニア振戦・ 小脳性振戦・Holmes振戦・心因性振戦など13疾患を系統的に鑑別します。
7ステップ / 13疾患鑑別
本ツールは臨床判断の補助を目的としており、確定診断を行うものではありません。
実臨床では総合的な臨床判断が必要です。
